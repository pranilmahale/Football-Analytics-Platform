# ⚽ Football Analytics Platform

A comprehensive **Football Analytics Platform** built with **Python**, **Streamlit**, and **PostgreSQL**. This project provides interactive dashboards, advanced visualizations, machine learning predictions, and AI-powered insights to analyze football players, clubs, leagues, competitions, transfers, and historical match data.

Designed as a portfolio-quality project, the platform demonstrates skills in data analytics, software engineering, machine learning, database management, and modern Python application development.

## ⚙ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Football-Analytics-Platform.git
cd Football-Analytics-Platform
```

### 2. Create a virtual environment

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/macOS**

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure environment variables

Create a `.env` file and configure your PostgreSQL database credentials and API keys.

### 5. Run the application

```bash
streamlit run app.py
```
